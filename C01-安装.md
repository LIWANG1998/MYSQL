## 2、brew

```shell
$ brew install mysql
```

## 3、配置

### 3.1. 环境变量

```shell
$ vim  ~/.bash_profile 
```

添加如下内容：

```shell
 PATH="$PATH":/usr/local/mysql/bin 
```

接下来需要source一下该文件，让刚刚设置的环境变量生效：

```shell
$ source ~/.bash_profile 
```

### 3.2. 生成配置文件

 在Mac系统中，安装完MySQL后，是没有自动生成MySQL的配置文件的，所以我们需要手动创建。

```shell
# 进入etc目录
$ cd /etc
# 创建my.cnf配置文件
$ touch my.cnf
# 修改文件权限
$ sudo chmod 664 my.cnf 
```
