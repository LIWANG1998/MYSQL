# # 配置

## 1. macOS

 在Mac系统中，安装完MySQL后，是没有自动生成MySQL的配置文件的，所以我们需要手动创建。

```shell
# 进入etc目录
$ cd /etc
# 创建my.cnf配置文件
$ touch my.cnf
# 修改文件权限
$ sudo chmod 664 my.cnf 
```

# # 链接

```shell
# 1. 链接数据库(输入密码)
$ mysql -u 用户名 -p 

# 2. 链接远程数据库
$ mysql -h ip地址 -u 用户名 -p
```

其他指令：

```js
// 1. 查看版本（连接后可以执行）
select version();

// 2. 显示当前时间（连接后可以执行）
select now();

// 3. 退出登录(断开连接)
quit 或 exit 或 \q
```
